2 文本修饰

2.1 文字样式

  • 加粗：使用 ** 或 __ 包裹需要加粗的文字。例如：**被加粗的文字** 或 __被加粗的文字__
  • 倾斜：使用 * 或 _ 包裹需要倾斜的文字。例如：*被倾斜的文字 或 _被倾斜的文字_
    • 建议：倾斜的汉字和未倾斜的汉字区别不大，又破坏了汉字的美感，不建议使用。
  • 删除线：使用 ~~ 包裹被标记为删除的文字。例如 ~~被删除的文字~~，效果 被删除的文字
  • 加粗且倾斜：**被加粗且 _被倾斜_ 的文字** 被加粗且 被倾斜 的文字
    • 注意：** 内部的 _ 前后需要一个空格，以使斜体效果发挥作用。

2.2 引用

  • 单行引用：使用 >。例如：> 被引用内容

  • 多行引用或代码块：使用 <!–￼1–>

  • 每行都缩进 4 格即可标记为代码块。（明显超过2行的代码量就不适合了）

  • 还可指明代码语言种类，以支持不同的代码高亮，如

<!–￼2–>
