长安十二时辰–大案牍术能拯救大唐吗？

得闻马亲王的《长安十二时辰》已上线播出，记得小说也才上市一二年的样子，居然就翻拍完成，速度之快令人惊讶。马亲王做客晓松奇谈时的“可不说”的说法，原来就是这个意思。

首先，这是一个蛮好的电视剧。我没有看过原著，先不乱下结论，说又是好本子撑起一台戏。至少开篇 2 分钟的开场镜头，就已经极为难得。

完蛋了，再按这个调调写下去，这篇东西就写不下去了。

我觉得这是一部很「妙」的剧。它披着古装戏的外壳，具备所有历史元素，但是讲了一个现代故事。具体这是一个什么样的现代故事，我就不展开了，如果看完全篇还没有意识，我也不再阐明。

全剧主要剧情就是：靖安司司丞李必能否带领靖安司，发挥靖安司的大案牍术和前“刑侦队长”张小敬的本领，及时抓获犯罪团伙狼卫。

靖安司的职责十分特殊，就是唐朝版本的 NSA。只是它只对内，且比较不合理地只负责长安的安危。

戡乱平镇曰靖，四方无事曰安。靖安司是朝廷新近秘设官署，统摄整个长安贼事策防。
主官何监，讳称执正，乃我朝三品秘书监，统管大唐明档秘档。靖安司在朝中地位，异乎寻常。

即便如此，靖安司也具备所有历史朝代的各类官署衙门所不具备的能力：大案牍术。

大案牍术

大案牍术是靖安司的一套情报采集和处理系统，主要组成部分有：

  • 数据的来源：朝中中书省、门下省、尚书省、吏部、户部、礼部、兵部、刑部、工部、御史台、大理寺、光禄寺，三省六部一台九寺五监机密要件，全在此处。

  • 剧中已经提到的数据源：

    • 假宅出租合约
    • 长安县一年内户籍变动
    • 西市署税案
    • 户部人口色役册
    • 工部土方建案

  • 数据的存储方式：竹册、卷轴、纸折本。

  • 数据的处理方式：人工查找

  • 数据的加速方式：徐宾大人直接一边摸串珠一边回忆

  • 数据的可视化：长安沙盘、长安地图

  • 实时数据的处理：望楼 + 武侯 + 旅贲军

    • 传输信道：
      • 信令灯符。（目前到白头的剧情，只知道有鼓点和信令）
      • 手持竹简的传令兵
    • 信道编码：以八卦卦象变化而来，设计繁复，短时间内难以记忆。望楼的通传武侯特地加以训练才可快速转译。

  • 数据的应用：「徐宾以大案牍术，按我的要求在长安城内筛选合格之人，选中了你和崔六郎，你们熟知长安明暗黑白规矩，通三教九流，懂多方语言，胜心重，有牵挂，想活。」

大案牍术的问题

剧情进行到现在，该系统已出现的问题：

  1. 数据错误：「此人于户籍上失踪，不知去向。」
  2. 数据录入方式不统一：「信安王军中闻姓者八十二人，烽燧堡二百二十二中闻姓者五人，幸存九人中闻姓者一人。可他与闻染父亲名字不同。」
  3. 数据丢失：「这些年，我唐卷宗案牍大不如前，都记得疏忽草率，语焉不详。」
  4. 数据源缺漏：「这些青楼目录狗屁用都没有，没有真的。……你们哪，能查到大唐的明档秘档，唯独查不出这平康坊的真实消息。」
  5. 数据处理能力跟不上领导的要求。
  6. 算法被质疑：「“那徐主事怎么知道，张小敬符合我的要求？”“不是徐某知道，是大案牍术知道。”」

大唐首席数据专家徐宾，耷拉了他的脑袋。

别伤心，徐宾，一千年后再来。

懒得写了，太无聊了。我要继续看剧去了。
